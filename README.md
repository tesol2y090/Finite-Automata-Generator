# Finite-Automata-Generator

    This project is part of Discrete Mathermetics, We build application that generate finite diagram from input and condition string.

### Prerequisites

NPM (Node Package Manangement)
Node.js


### Installing


```
Npm install
```

```
npm start
```

or

```
yarn start
```
